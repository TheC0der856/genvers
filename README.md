# genvers
The app can calculate KBAs for single threatened and range restricted species (criteria A1 and B1). In addition to the packages ADEGENET, POPPR, and VEGAN the app GENVERS is using packages linked to a shinyapp creation. The app was published via shinyapps.io by Posit for free. It is possible to upload datasets as .csv, .dat, .gen, .gtx, .str, or .stru files. Example datasets can be downloaded for each format and afterwards fed into the app. The references belonging to each example dataset can be retrieved from the website. Distinct genetic diversity for each site can be displayed as a downloadable table. The most important function of the app is the identification of KBAs after selecting the KBA criteria appropriate for the dataset. A user manual can be downloaded from the app’s website in case of questions. However, it cannot be used by multiple users at the same time and permits partitioning of up to 1 GB of memory and 8 GB of RAM. When GENVERS is executed for large datasets (like most SNP datasets) the user may disconnect from the server after a certain time without results. For smaller datasets (like most microsatellite datasets) this is not relevant.
